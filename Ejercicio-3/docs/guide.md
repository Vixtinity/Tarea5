# Guía de Uso

## Cómo hacer una reserva

1. **Selecciona una Hora**:
   En la parte superior, haz clic en una de las horas disponibles (ej: 13:00, 14:00).
   El mapa se actualizará para mostrar el estado de las mesas en esa hora.

2. **Selecciona una Mesa**:
   Busca una mesa Libre (color verde).
   Haz clic en ella. Se marcará en azul como seleccionada.
   _Nota: No puedes seleccionar mesas Ocupadas (rojo)._

3. **Completa el Formulario**:
   Aparecerá un formulario a la derecha (o debajo en móviles).
   Ingresa:

   - **Nombre**: Obligatorio.
   - **Email**: Debe ser válido.
   - **Comensales**: Número de personas (no debe superar la capacidad de la mesa).

4. **Confirmar**:
   Haz clic en "Confirmar Reserva".
   Si todo es correcto, recibirás una alerta de confirmación y la mesa pasará a estar ocupada.

## Accesibilidad

- Puedes usar la tecla `TAB` para navegar por los horarios y mesas.
- Usa `ENTER` para seleccionar.
- Los lectores de pantalla anunciarán el estado y capacidad de cada mesa.
